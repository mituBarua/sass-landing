# SaaS Landing Page — Figma to React

Pixel-perfect conversion of a Figma design to React.

## Live Demo
(https://sass-landing-jcsf.vercel.app/)

## Sections
- Navbar with mobile menu
- Hero with email input
- Reviews
- How it Works
- Analytics
- Feature section
- Pricing with hover effects
- Testimonial slider with auto-slide
- Footer

## Built With
- React + TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Vercel (deployment)
